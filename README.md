## How to use?

1. Clone this repository

    ```bash
    git clone https://github.com/altanbgn/dotfiles
    cd dotfiles
    ```

2. Install GNU Stow

    ```bash
    # I use arch btw
    yay -S stow
    ```

3. Uncomment the lines in [`install.sh`](./install.sh) of dotfiles you want to install
4. Execute `./install.sh`.
